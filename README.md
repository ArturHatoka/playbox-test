# Playbox Test (Cocos Creator)

Тестовый проект по разработке playable-клона референса:
https://playbox.play.plbx.ai/playoff/runner

## Стек
- Cocos Creator (2D)
- JavaScript/TypeScript (в зависимости от выбранной конфигурации проекта)

## Цель
Собрать клон playable-креатива с нуля на Cocos Creator и подготовить финальную сборку в формате одного HTML-файла с ограничением по размеру до 5 МБ.

## Требования из задания
- Исходный код хранится в репозитории.
- Финальный билд: один HTML-файл.
- В HTML должны быть встроены ассеты, скрипты и стили.
- Размер итогового бандла: не более 5 МБ.
- История разработки должна содержать несколько осмысленных коммитов.

## Как начать
1. Откройте проект в Cocos Creator.
2. Настройте стартовую сцену и базовые игровые объекты.
3. Реализуйте core-геймплей по референсу.
4. Подготовьте single-file HTML сборку.
5. Проверьте ограничение размера бандла.
